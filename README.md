# TACTypes
Definiciones de tipos para el tipo de dato `TAC`, implementación común para los grupos de lenguajes de programación 3 sept-dic 2021

```
assign ADDR X

goto LABEL
goif LABEL X

eq  ADDR X Y
neq ADDR X Y
lt  ADDR X Y
leq ADDR X Y
gt  ADDR X Y
geq ADDR X Y

add  ADDR X Y
sub  ADDR X Y
mult ADDR X Y
div  ADDR X Y
mod  ADDR X Y

param ADDR
call  ADDR X
```
